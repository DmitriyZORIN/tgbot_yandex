В наше время, нельзя забывать о безопасности своих данных и чтобы не попадаться на фишинговые и вредоносные ссылки я написал телеграмм бота для проверки ссылок на безопасность

В боте имеется регистрация и подписка(функция проверки ссылок специально открыта и без подписки, чтобы не усложнять тестирование, но вы можете проверить её работоспособность с помощью тестовой карты: 1111 1111 1111 1026, 12/22, CVC 000)

В боте используется три способа проверки:
проверка ssl сертификата
проверка на redirect
проверка с помощью api virustotal
ссылки для проверки, так же вы конечно можете проверить и свои ссылки
(https://www.python.org
http://m.bibika.ru/
https://bit.ly/1oJrmXs
http://stevensandgoldwyn.com/)
